## Build Commands
- `zig build` - Build native + run unit tests
- `zig build test` - Build native + run integration tests
- `zig build all` - Build all platforms (no tests)

## Directory Structure
- `src/` - Source code
- `tests/` - Test files
- `docs/` - Documentation
- `scripts/` - Utility scripts
- `build_tools/` - Build framework

## Dependencies
